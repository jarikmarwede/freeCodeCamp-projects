This is an Image Search Abstraction Layer built during a web development course on freeCodeCamp.org

The API has two query urls:

  1. /.netlify/functions/Image_Search_query?query=<yourSearchTerm>&offset=<pageNumber> (e.g. /.netlify/functions/Image_Search_query?query=cats&offset=1. The default offset is 1)
  2. jm-freecodecamp-projects.herokuapp.com/image-search-abstraction-layer/latest/imagesearch
