This is a URL Shortener Microservice built during a web development course on freeCodeCamp.org.

The API can be used via a POST request to jm-freecodecamp-projects.herokuapp.com/url-shortener-microservice/shorturl/new with the argument "url" set to the URL you want to have shortened. It will then respond with:

  • "original_url" set to your URL and "short_url" set to the shortened version
  • Or with an "error" argument set to the corresponding error message
